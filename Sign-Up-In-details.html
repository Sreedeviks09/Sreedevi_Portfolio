Signup and Signin System Using Python-Django

Signup and Signin System Using Python-Django This project is a web-based authentication system built with Python-Django that includes functionality for user registration (signup) and user login (signin). It also includes features to manage user sessions securely. Additionally, the system is integrated with a Django superuser for admin-level control and provides the option for users to update their profiles via a portfolio feature.

Technologies Used: Backend: Python, Django Framework Frontend: HTML, CSS, Bootstrap Database: Admin(Super User) Version Control: Git/GitHub (for version control and project collaboration)

Features User Signup: Allows new users to register by providing essential details such as username, email, and password. Passwords are securely hashed using Django's inbuilt authentication framework. User Signin: Authenticated users can log in to access personalized content. Implements session management to ensure secure user login. Superuser Functionality: Admins can create a superuser account using Django's createsuperuser command. The superuser has full control over managing user accounts, resetting passwords, and viewing user activity in the admin panel. Portfolio Management: Users can upload and update their portfolio information. Portfolios may include a bio, professional details, profile picture, and links to personal projects. Security: Implements form validation to prevent malicious input. Ensures proper session expiration and secure logout functionality. Responsive Design: The user interface is built using HTML, CSS, and Bootstrap, making it user-friendly and responsive.

Project Structure: views.py: Contains the logic for handling requests and rendering the appropriate HTML templates. Views for IndexPage, SignUpPage, SignInPage and Signout. urls.py: URL routing defines how requests are mapped to views (URLs for Index, SignUp, SignIn and Signout). models.py: Defines the Users model with fields like username, email, and date that password and conformpassword core attributes. templates: index.html: Displays a list of welcome form and logout link is their. signin.html: Displays a loginpage. signup.html: Provides a form to Signuppage. files: CSS (style.css): Custom styles to make the application visually appealing.

Key Learning Outcomes: Django Fundamentals: Learned how to build a web application with Django’s MVT (Model-View-Template) architecture. Mastered Django models, forms, views, and templates for creating dynamic web applications. Database Handling: Used Django’s ORM (Object-Relational Mapping) for database operations, including adding, updating, and deleting records. Demonstrated the ability to integrate the app with a relational database (SQLite). Frontend Development: Created interactive and responsive forms using HTML and Bootstrap for the user interface. Integrated custom styles in CSS for better UI/UX. Version Control: Used Git for version control and GitHub for managing and sharing the code.

This project demonstrates expertise in full-stack web development with Django, focusing on e-commerce functionality and responsive design. It showcases skills in back-end development, front-end design, and database management.
