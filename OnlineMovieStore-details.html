Online Movie Store

Online Movie Store with Django The Online Movie Store is a web application designed to provide users with a seamless experience of browsing, searching, and purchasing movies. This platform is built using the Django framework, combined with responsive front-end technologies like HTML, CSS, and Bootstrap for a user-friendly interface. The database management system ensures secure storage and retrieval of movie and user data.

Technologies Used: Backend: Python, Django Framework Frontend: HTML, CSS, Bootstrap Database: Admin(Super User) Version Control: Git/GitHub (for version control and project collaboration)

Creation: Users can add new tasks with details like task name, priority, and due date. A form is provided for users to input task details, which are then stored in a database. Listing: All the tasks are displayed on the homepage. Each task displays its name, priority, and due date. Update: Users can update existing tasks by modifying the task name, priority, or due date. A user-friendly form is provided for editing tasks, with pre-filled values that can be updated. Deletion: Users can delete tasks they no longer need. A confirmation page is shown before deletion to prevent accidental removal of tasks. Management: CRUD operations are implemented (Create, Read, Update, Delete) allowing the user to manage tasks effectively. The app ensures that the userâ€™s task data is safely saved in the database and can be retrieved whenever required.

Project Structure: views.py: Contains the logic for handling requests and rendering the appropriate HTML templates. Views for adding a new movie (Movieadd), deleting a movie (delete), and updating a movie (update). urls.py: URL routing defines how requests are mapped to views (URLs for home, movie deletion, movie update). models.py: Defines the moviestore model with fields like name, priority, and date that represent a task's core attributes. forms.py: A Django ModelForm is used to handle movie creation and updates via a form interface. templates: home.html: Displays a list of movie and a form to add new movie. delete.html: Displays a confirmation form before deleting a movie. edit.html: Provides a form to update an existing movie. base.html: Base template containing common HTML structure like navigation, header, and footer. Static files: CSS (style.css): Custom styles to make the application visually appealing.
